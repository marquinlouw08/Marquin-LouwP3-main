# Marquin-LouwP2

## Overview

This repository contains a simple static website project with the following pages:

- `index.html`
- `about.html`
- `product.html`
- `contact.html`
- `enquiry.html`

The site uses `style.css` for layout and appearance, and `script.js` for any interactive behaviour.

## Files

- `index.html` — home page
- `about.html` — information about the project or business
- `product.html` — product details or listing
- `contact.html` — contact information page
- `enquiry.html` — enquiry/contact form page
- `style.css` — global stylesheet
- `script.js` — JavaScript for dynamic page behavior

## Usage

Open the HTML files in a web browser to view the website locally. No build step or web server is required for basic browsing.
